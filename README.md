# Name-isname Taskctl

## How do I install these formulae?

`brew install name-isname/taskctl/<formula>`

Or `brew tap name-isname/taskctl` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "name-isname/taskctl"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
